# SB RFI 策略：preopen 1B vs 2B 对照（2026-06-02）

> 工作笔记 / 原始数据记录。对应 `docs/six_max_nlhe_target.md` S4 续 ⑥⑦「SB 欠训练 vs 抽象天花板」未决问题。

## 是什么

- **SB RFI** = preflop fold 到 SB（UTG→BTN 全弃），SB 首个行动 vs BB 时的 169 手策略。
- **来源**：`preopen` 变体（= nolimp + preflop `{0.5,1}` 加 2.25BB 开池档）AWS 5B run，~2.1B 处暂停的 **1B / 2B** checkpoint。dump 存 vultr `artifacts/run_6max_s4_preopen/preflop_rfi_{1B,2B}.md` 的 SB 段。
- **列**：`L`=Limp（完成到 1BB）/ `2.25`=Raise 0.5x（2.25BB 开池）/ `3.5`=Raise 1x（3.5BB 开池）/ `F`=Fold。AllIn≈0（仅 JJ 0.001）略，故每行和 ≈1。
- **用途**：SB 随训练量（1B→2B）是否收敛——回答「SB 残留 17% 翻转是欠训练还是抽象天花板」。

## 结论（克制，不下定论）

- **非盲位 1B 已收敛**（漂移 ~0.10、中位 0），且范围放宽近 GTO（BTN raise 44–45%）——这部分训练充分。
- **SB 在 2B 仍剧烈 churn**（漂移 0.284）：范围边界（哪些手玩）基本稳，但**档位分配（limp/2.25/3.5）大幅翻动**（AKs 三档混合→几乎纯 2.25；Q8s limp .99→.49/.26/.25；T4s/T2s/55/KJs/K9o 翻档）。
- **故不能判 SB 是「欠训练」还是「抽象天花板（limp 选项造的天然宽混合区，EV 太平永远 churn）」**——2B 处它还在动，证据不足。若非要倾向：还在动说明未收敛，5B 可能继续磨稳（也可能永远 churn）。
- **要分清**：把 preopen 跑满 5B（新 LCFR run）看 SB 漂移是否随训练量衰减到非盲位水平（~0.10）；或加「删 SB limp」变体直接消除该混合区做对照。
- GTO 参照：无 rake solve 的 SB limp 本就低（≈ HU 按钮 raise-or-fold）；判好坏看**单调性**（AA/KK 该 raise ~100%）而非 limp 频率——本 SB 仍 limp AA 35%、K4s limp 100% 而 K5s 加注，属噪声痕迹。
